# yome-vs-shuto
# 👰 嫁 VS 姑 バトル 👵

スマホでケータイ格闘ゲーム！嫁と姑の壮絶バトル。夫と義父はおどおどしながら見守っています。

## 🎮 操作方法

| 操作 | 内容 |
|------|------|
| スライド（下エリア） | キャラクター左右移動 |
| 👊 攻撃ボタン | パンチ（近距離必須） |
| 🛡️ 防御ボタン | 長押しでガード（攻撃を弾き返す） |
| 🦋 ジャンプボタン | ジャンプ |
| ⚡ ビームボタン | 必殺ビーム（ゲージMAX時のみ） |

## ⚡ ビームシステム

- パンチを**8発**当てるとゲージがMAXに
- ゲージMAXで「READY!」が光る → ⚡ビームボタンが使える
- ビームはパンチの**2倍のダメージ**
- パンチは**15発**で相手を倒せる威力

## 🎨 背景シーン（ランダム）

1. **🍳 台所バトル** — 火の粉飛ぶキッチン。旦那が冷蔵庫の影でおどおど
2. **🛋️ リビングバトル** — ソファでTV観てたら巻き込まれた義父
3. **🌸 庭バトル** — 桜散る中の戦い。旦那が木の裏から震えて見てる

## 🚀 GitHub Pagesでの公開方法

1. GitHubでリポジトリ作成（例: `yome-vs-shuto`）
2. `index.html` をアップロード
3. Settings → Pages → Source: `main` ブランチ → Save
4. `https://yourusername.github.io/yome-vs-shuto/` でプレイ可能！
